API Example

用户拿到 key 后这样调用

模板变量在前端替换，生成最终 prompt 后，直接走 Flatkey 的 OpenAI 兼容生图接口。

curl https://api.flatkey.ai/v1/images/generations \
  -H "Authorization: Bearer $FLATKEY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "把模板变量替换后的最终提示词",
    "size": "1536x1024"
  }'
